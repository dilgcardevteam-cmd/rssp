# System Manual Files

This folder contains role-based manual content rendered inside the application.

## Files
- `applicant-manual.html`
- `admin-manual.html`
- `hr-division-manual.html`
- `viewer-manual.html`

## Screenshot storage
Place screenshot images under `public/docs/manual/assets/<role>/`.

Role folders:
- `public/docs/manual/assets/applicant/`
- `public/docs/manual/assets/admin/`
- `public/docs/manual/assets/hr_division/`
- `public/docs/manual/assets/viewer/`

Use numbered image names to keep order consistent with each manual.
