Need a complete admin workflow from vacancy posting to reporting?

This guide is for superadmin and admin users. Follow each section in order when onboarding a new admin user.

Admin Dashboard

Steps:

  1. Open dashboard and check pending workloads.
  2. Review summary cards and quick links.
  3. Proceed to the target module.

Vacancies Management

Steps:

  1. Open Vacancies Management.
  2. Add COS or Plantilla vacancy.
  3. Complete details and save.

Add COS Vacancy

Steps:

  1. Open Vacancies Management.
  2. Select Add COS.
  3. Fill in vacancy details and save.

Add Plantilla Vacancy

Steps:

  1. Open Vacancies Management.
  2. Select Add Plantilla.
  3. Fill in plantilla details and save.

Add COS from Positions

Steps:

  1. Open Positions utility.
  2. Select a position then add as COS vacancy.
  3. Review details and save.

Add Plantilla from Positions

Steps:

  1. Open Positions utility.
  2. Select a position then add as Plantilla vacancy.
  3. Review details and save.

Applicants Management

Steps:

  1. Open Applicants Management.
  2. Filter by vacancy and status.
  3. Review applicant profile and update status.

Exam Management

Steps:

  1. Open Exam Management.
  2. Select vacancy exam.
  3. Set schedule, monitor, and finalize updates.

Utilities Tools

The Utilities Tools menu is your entry point for Programs, Positions, Signatories, Activity Logs, Reports, and Backup/Restore.

Steps:

  1. Open Utilities Tools from sidebar.
  2. Select the specific utility module needed for your task.
  3. Complete changes and verify updates in the target module screen.

Reports

Steps:

  1. Open Reports.
  2. Apply filters.
  3. Export report output.

Activity Log

Steps:

  1. Open Activity Log.
  2. Filter date, module, and actor.
  3. Review audit trail entries.

Signatories

Steps:

  1. Open Signatories.
  2. Add or update signatory data.
  3. Save and verify usage in outputs.

User Management (superadmin)

Steps:

  1. Open User Management.
  2. Create or update accounts/roles.
  3. Save and verify account access.

Applicant Records (superadmin)

Steps:

  1. Open Applicant Records.
  2. Search target applicant.
  3. Open record history and perform allowed actions.

Programs Management

Steps:

  1. Open Utilities Tools then Programs.
  2. Add or update program presets.
  3. Save and validate listing.

Position Management

Steps:

  1. Open Utilities Tools then Positions.
  2. Review and update position records.
  3. Save changes.

Backup and Restore

Steps:

  1. Open Utilities Tools then Backup and Restore.
  2. Create backup or run restore workflow.
  3. Confirm completion and review logs.

HR Admin Access

Steps:

  1. Open User Management.
  2. Assign or update HR Division vacancy access.
  3. Save and validate access scope.
