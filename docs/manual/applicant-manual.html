Need a step-by-step flow from account setup to application tracking?

This applicant manual is arranged in the exact order most users should follow.

Dashboard

Steps:

  1. Open Dashboard after login.
  2. Check alerts, reminders, and profile completeness.
  3. Continue to the next required module.

Account Edit

Steps:

  1. Open profile menu.
  2. Select account settings.
  3. Update profile details and save.

Job Vacancies

Steps:

  1. Open Job Vacancies from sidebar.
  2. Use search/filter to find a vacancy.
  3. Open the vacancy details page.

Job Description and Apply

Steps:

  1. Review qualifications and requirements.
  2. Click Apply.
  3. Continue to PDS and document requirements.

Personal Data Sheet (C1)

Steps:

  1. Open the Personal Information section.
  2. Fill in personal details.
  3. Click Save before moving to the next section.

Personal Data Sheet (C2)

Steps:

  1. Open Work Experience section in PDS.
  2. Add entries and review dates.
  3. Save changes.

Personal Data Sheet (C3)

Steps:

  1. Open Learning and Development.
  2. Add training and seminars.
  3. Save changes.

Personal Data Sheet (C4)

Steps:

  1. Open Other Information.
  2. Complete declarations and references.
  3. Save changes.

Personal Data Sheet (C5)

Steps:

  1. Open the final submission section.
  2. Upload required files if prompted.
  3. Finalize when all required fields are complete.

Work Experience Sheet (WES)

Steps:

  1. Open Work Experience Sheet.
  2. Add entries relevant to the position.
  3. Save changes.

Document Gallery

Steps:

  1. Open profile document gallery.
  2. Review uploaded files.
  3. Replace files only when updates are needed.

Download PDS

Steps:

  1. Open Download Documents.
  2. Preview and download your PDS output.

Download WES

Steps:

  1. Open Download Documents.
  2. Preview and download your WES output.

My Applications

Steps:

  1. Open My Applications.
  2. Select your submitted application.
  3. Check status, remarks, and next instructions.

Notifications and Profile Menu

Steps:

  1. Open Notifications to review updates.
  2. Open profile menu for account settings.
  3. Logout only after saving all form changes.
