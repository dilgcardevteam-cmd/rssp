Need to monitor exams without changing admin settings?

This manual is for the viewer role.

Exam Management Landing Page

Steps:

  1. Open Exam Management.
  2. Select an ongoing exam session.

Exam Monitor

Steps:

  1. Open monitor view.
  2. Track session status and violations.
  3. Keep records for admin review.

Applicant Answer View

Steps:

  1. Open answer view for a selected applicant.
  2. Review responses for monitoring purposes.
  3. Escalate issues to admin when needed.

Access Scope

  1. Viewer can monitor and view exam information.
  2. Viewer cannot score, finalize, or edit admin-only configuration.
