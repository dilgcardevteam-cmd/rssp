# Manual Screenshot Assets

Put all manual screenshots in this folder so they are available inside the system.

## Folder structure
- `public/docs/manual/assets/applicant/`
- `public/docs/manual/assets/admin/`
- `public/docs/manual/assets/hr_division/`
- `public/docs/manual/assets/viewer/`

## Naming recommendation
Use numbered files to match the module order in docs/manual markdown files.
Example: `01-dashboard.png`, `02-vacancies.png`.

## Image tips
- PNG format preferred.
- Keep consistent resolution when possible.
- Remove or blur sensitive personal data before uploading screenshots.
