# Manual Screenshot Assets

Store screenshots used by in-system manuals in this folder.

## Role folders
- `applicant/`
- `admin/`
- `hr_division/`
- `viewer/`

## Naming suggestion
Use ordered names to match each manual section:
- `01-dashboard.png`
- `02-module-name.png`
- `03-action-name.png`

Images are served from URLs like:
- `/docs/manual/assets/applicant/01-dashboard.png`
